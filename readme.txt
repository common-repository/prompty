== Description ==

This plugin adds the prompt-like effect on the pre tag contents in your post.
It is useful for showing command prompt or unix shell type of samples.

Setup is just easy. You can just add the `prompty` CSS class
to the pre tags that you want to have the effect in your post.
Then, you will see the `Play` button on each pre tag contents when you view it,
and you can click it to see the "prompty" effect. 

Demo video is available at <a href="http://www.youtube.com/watch?v=jf0UMF5nAQk">YouTube</a>.

== Installation ==

1. Download, unzip and upload the `prompty` folder to the `/wp-content/plugins/` directory.
2. Login to your WordPress administration screen and activate the plugin. 

or

1. Click `Add New` and search "prompty" from the WordPress administration screen and install it.
